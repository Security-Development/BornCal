ver(sion)3은 기본코드(ver1)와 ver2의 코드들에서 사용되던 TCP 프로토콜을 이용하는것에서 벗어나 UDP 프로토콜 기반 통신 코드를 제작하기 위해서 만들어졌습니다.
이는 성공적으로 마쳤으며 이후 ver2의 u_cl.c 와 u_sv.c 의 코드 이해와 기능 개선에서 큰 도움이 됐습니다.

ver(sion)3 was created to create UDP protocol-based communication code, away from using the TCP protocol used in the codes of base code (ver1) and ver2.
This was successful and helped us improve our code understanding and functionality in ver2's u_cl.c and u_sv.c.
