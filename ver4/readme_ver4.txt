ver4는 ver3에서 만들어진 udp 기반 통신 기능에 자체 제작 구조체를 이용하여 패킷을 주고받는 코드를 제작하기 위해 만들어졌습니다.
하지만 ver2 의 u_cl.c 와 u_sv.c 코드의 습득을 통해 프로젝트 자체가 무산됐습니다.
위의 u_cl.c 와 u_sv.c는 완벽하게 성공적으로 설정된 자체 제작 패킷이 적용됐기 때문에 ver4 버전과 디렉터리는 폐기됐습니다.
현재의 ver4 디렉터리는 기록저장을 목적으로써 남겨있습니다.


ver4 was created to create code for sending and receiving packets using its own built-in structure for udp-based communication function created in ver3.
However, the project itself fell through with ver2's acquisition of u_cl.c and u_sv.c codes.
The above u_cl.c and u_sv.c have been dropped ver4 versions and directories because of the fully successfully configured self-made packets.
The current ver4 directory is reserved for recording purposes.
